# 茶言觀色·客戶流失風險評估器

Tea Observation Customer Churn Risk Assessor

## 專案概述

「茶言觀色·客戶流失風險評估器」是一個專為茶飲店設計的單頁網頁應用程式，幫助管理層快速評估客戶流失風險，並提供數據驅動的留客建議。

## 功能特色

- **即時風險評估**：輸入客戶基本資料，立即獲得流失風險評估
- **數據驅動分析**：基於實際業務數據分析得出的規則進行評估
- **視覺化結果**：直觀的風險等級、流失機率和行動建議顯示
- **響應式設計**：支援桌面、平板和手機裝置
- **使用者友善介面**：簡潔直觀的操作介面，無需技術背景

## 使用方式

### 輸入客戶資料
1. **年齡** (Age)：輸入客戶年齡 (18-80歲)
2. **每月到訪次數** (Monthly Visits)：輸入客戶每月到訪次數 (0-30次)
3. **最近一次到訪天數** (Days Since Last Visit)：輸入客戶最近一次到訪至今的天數 (0-365天)
4. **滿意度評分** (Satisfaction Score)：選擇客戶滿意度評分 (1-5分，5分為最高)

### 查看評估結果
- **風險等級**：顯示低、中、高三種風險等級
- **流失機率**：顯示客戶流失的百分比機率
- **行動建議**：提供針對性的留客策略建議
- **風險因素分析**：分析各因素對流失風險的影響程度

## 風險評估規則

### 高風險條件（符合任一項）
1. 最近到訪天數 > 60天 (流失率：45.2%)
2. 滿意度 ≤ 2分 (流失率：42.9%)
3. 每月到訪次數 ≤ 3次 (流失率：38.7%)

### 中風險條件（不符合高風險，且符合任一項）
1. 最近到訪天數 30-60天
2. 滿意度 = 3分
3. 每月到訪次數 4-6次

### 低風險條件（需同時滿足所有條件）
1. 最近到訪天數 < 30天
2. 滿意度 ≥ 4分
3. 每月到訪次數 ≥ 7次

### 流失機率估算
- 高風險：~42%
- 中風險：~28.5%
- 低風險：~15%

## 快速開始

### 方法一：直接開啟
1. 下載專案檔案
2. 使用瀏覽器開啟 `index.html` 檔案

### 方法二：本地伺服器
```bash
# 使用 Python 啟動簡單 HTTP 伺服器
python -m http.server 8000

# 或使用 Node.js 的 http-server
npx http-server
```

然後在瀏覽器中訪問 `http://localhost:8000`

## 開發指南

### 修改風險評估規則
風險評估邏輯位於 `script.js` 檔案中的 `calculateRiskLevel` 函數。您可以根據業務需求調整規則。

### 自訂樣式
樣式變數定義在 `style.css` 檔案的 `:root` 區塊中，您可以修改顏色、間距和字體等變數。

### 擴充功能
1. **歷史記錄**：添加 localStorage 儲存評估記錄
2. **批量評估**：添加 CSV 檔案上傳功能
3. **報表匯出**：添加 PDF 或 Excel 匯出功能
4. **API 整合**：連接後端資料庫進行更精準的預測
